Docker搭建分布式大数据环境之CentOS的一些初始化设置

发表于 2017-10-09 | | 阅读次数:

前言

在Docker搭建分布式大数据环境之准备工作中我们成功将CentOS系统写入到移动硬盘中。进入之后机会发现在这个最小系统里面真的是什么都没有，ifconfig这些基本的命令也没有，不知道自己的ip，也ping不通外网，很蛋疼啊。接下来我们就来进行一些操作让我们的CentOs系统真正的飞起来。

网络设置

网络是必须的，一般的设置都是那种开机自动随机ip，这个病不符合我们的需求，因为后面的操作基本上都是使用xshell远程操作（我是这样的，因为在黑漆漆的CentOS里面搞事情确实不爽，有时候需要查一些资料什么的就非常的蛋疼），因此需要的是固定ip。

编辑CentOS的网络配置文件

cd  /etc/sysconfig/network-scripts

进入到目录后会发现一个ifcfg-eth0文件

vi ifcfg-eth0

将文件中如下几个字段改成如下的内容：

BOOTPROTO=static #dhcp改为static   
ONBOOT=yes #开机启用本配置  
IPADDR=192.168.1.62 #静态IP，根据自己需要设定  
GATEWAY=192.168.1.1 #默认网关  
NETMASK=255.255.255.0 #子网掩码  
DNS1=192.168.1.1 #DNS 配置 
DNS2=8.8.8.8

然后还需要修改/etc/sysconfig/network文件增加网关如下：

GATEWAY=192.168.1.1

重启网络服务：

systemctl restart network

完成后使用如下命令查看IP:

ip address

需要说明的是一般情况下我们使用的是ifconfig来查看ip，但是由于这些东都没有装所以使用原生的命令即可。这时候就可以试着ping自己的网关192.168.1.1，以及ping百度，如果能ping通就万事大吉，接下来就直接通过xshell连接就行。连上之后首先做的事就是要关闭防火墙，因为后面要干很多坏事：

systemctl stop firewalld.service #关闭防火墙
systemctl disable firewalld.service #禁止firewall开机启动

更新yum源

推荐使用阿里的yum源，具体步骤如下

安装wget

yum install -y wget

下载repo文件

wget http://mirrors.aliyun.com/repo/Centos-7.repo

备份并替换系统的repo文件

cp Centos-7.repo /etc/yum.repos.d/ 
cd /etc/yum.repos.d/ 
mv CentOS-Base.repo CentOS-Base.repo.bak 
mv Centos-7.repo CentOS-Base.repo

执行yum源更新命令

yum clean all 
yum makecache 
yum update

装一些基本软件

yum install -y vim 
yum install net-tools

ok,有网，一些基本的软件有了，基本的环境就差不多了，接下来就可以直接开始进入主题了
